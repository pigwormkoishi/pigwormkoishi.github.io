为个人网站添加必应搜索

使用 Hugo 构建静态博客时，站内搜索一直是个绕不开的话题。很多人会选择本地索引库（如 Fuse.js 或 Pagefind），但这往往会增加编译时间或需要额外的配置。

对于个人博客而言，我更倾向于一种**“少即是多”**的轻量级方案：直接调用必应 (Bing) 搜索。零配置、零维护成本、完全白嫖搜索引擎的算力。

为了不破坏 Blowfish 主题原有的极简导航栏，我抛弃了传统死板的输入框，利用 Tailwind CSS 和原生 JS，手搓了一个带有呼吸感、高斯模糊以及实时下拉联想词的隐藏式全网搜索框。

设计与交互体验
#

默认状态下居中显示，宽度克制。背景采用 backdrop-filter: blur(8px) 的毛玻璃效果，完美融入网站的深色/浅色模式，不喧宾夺主。 当光标点击聚焦时，它会伴随丝滑的贝塞尔曲线（cubic-bezier）动画向两侧拉长展开，边框亮起薄荷绿（#98FB98）的高光微晕，给用户明确的输入反馈。 采用 JSONP 跨域请求拦截必应官方 API，配合 250ms 防抖处理。打字停顿瞬间，输入框下方会丝滑弹出带有磨砂玻璃背景的热词下拉菜单，鼠标悬停时泛起薄荷绿交互。 点击联想词或按下回车后，自动在新标签页打开全网搜索结果，搜完自动清空并隐藏。

核心代码实现
#

以 Blowfish 主题为例，我们需要利用 Hugo 的优先级机制覆写顶部的导航栏文件，避免直接修改主题源码。

第一步： 在你的博客根目录新建路径 layouts/partials/header/，并将主题目录下的 themes/blowfish/layouts/partials/header/basic.html 完整复制过来。

第二步： 打开你刚复制出来的 basic.html，在代码中部的区域（Logo 和右侧菜单之间的位置）插入以下核心代码：

<style>
    .bing-search-wrapper { 
        display: flex; 
        align-items: center; 
        justify-content: center; 
        width: 100%; 
    }
    
    /* 相对定位锚点，用于对齐下拉框 */
    .search-box-container {
        position: relative;
        display: inline-block;
    }

    .bing-search-input {
        background: rgba(150, 150, 150, 0.1); 
        border: 1px solid rgba(150, 150, 150, 0.2);
        border-radius: 20px;
        padding: 5px 16px; 
        font-size: 13.5px;
        color: inherit; 
        outline: none;
        width: 240px; 
        transition: all 0.4s cubic-bezier(0.25, 0.8, 0.25, 1);
        backdrop-filter: blur(8px);
        -webkit-backdrop-filter: blur(8px);
    }
    .bing-search-input:focus {
        width: 340px; 
        border-color: #98FB98; 
        box-shadow: 0 0 8px rgba(152, 251, 152, 0.3);
        background: rgba(150, 150, 150, 0.15);
    }
    .bing-search-input::placeholder { color: inherit; opacity: 0.5; }

    /* 下拉联想菜单 */
    .bing-suggest-box {
        position: absolute;
        top: 110%; 
        left: 0;
        width: 100%;
        background: rgba(255, 255, 255, 0.1);
        backdrop-filter: blur(12px);
        -webkit-backdrop-filter: blur(12px);
        border: 1px solid rgba(150, 150, 150, 0.2);
        border-radius: 12px;
        box-shadow: 0 8px 16px rgba(0,0,0,0.1);
        overflow: hidden;
        display: none; 
        z-index: 9999;
    }
    
    .dark .bing-suggest-box { background: rgba(30, 30, 30, 0.6); }
    .light .bing-suggest-box { background: rgba(255, 255, 255, 0.6); }

    .bing-suggest-item {
        padding: 8px 16px;
        font-size: 13px;
        color: inherit;
        cursor: pointer;
        transition: background 0.2s ease;
        white-space: nowrap;
        overflow: hidden;
        text-overflow: ellipsis;
    }
    .bing-suggest-item:hover {
        background: rgba(152, 251, 152, 0.3);
    }
</style>

<div class="flex-1 justify-center hidden md:flex"> 
    <div class="bing-search-wrapper">
        <div class="search-box-container">
            <form onsubmit="doBingSearch(event)" style="margin: 0; padding: 0;">
                <input type="text" id="bing-keyword" class="bing-search-input" placeholder="Bing 全网搜索..." autocomplete="off">
            </form>
            <div id="bing-suggest-box" class="bing-suggest-box"></div>
        </div>
    </div>
</div>

<script>
    const searchInput = document.getElementById('bing-keyword');
    const suggestBox = document.getElementById('bing-suggest-box');
    let debounceTimer = null;

    function doBingSearch(e, directKeyword = null) {
        if(e) e.preventDefault(); 
        let keyword = directKeyword || searchInput.value;
        if (keyword.trim() !== "") {
            let bingUrl = "[https://www.bing.com/search?q=](https://www.bing.com/search?q=)" + encodeURIComponent(keyword);
            window.open(bingUrl, '_blank'); 
            searchInput.value = ''; 
            searchInput.blur(); 
            suggestBox.style.display = 'none'; 
        }
    }

    searchInput.addEventListener('input', function() {
        clearTimeout(debounceTimer);
        let keyword = this.value.trim();
        
        if (keyword === "") {
            suggestBox.style.display = 'none';
            return;
        }

        debounceTimer = setTimeout(() => {
            let script = document.createElement('script');
            script.src = `https://api.bing.com/osjson.aspx?query=${encodeURIComponent(keyword)}&JsonType=callback&JsonCallback=renderBingSuggest`;
            document.body.appendChild(script);
            script.onload = () => document.body.removeChild(script);
        }, 250);
    });

    window.renderBingSuggest = function(data) {
        let suggests = data[1] || [];
        if (suggests.length === 0) {
            suggestBox.style.display = 'none';
            return;
        }

        suggestBox.innerHTML = ''; 
        suggests.slice(0, 6).forEach(word => {
            let div = document.createElement('div');
            div.className = 'bing-suggest-item';
            div.textContent = word;
            div.onclick = function() {
                searchInput.value = word;
                doBingSearch(null, word);
            };
            suggestBox.appendChild(div);
        });
        suggestBox.style.display = 'block'; 
    };

    document.addEventListener('click', function(e) {
        if (e.target !== searchInput && e.target !== suggestBox) {
            suggestBox.style.display = 'none';
        }
    });
</script>
